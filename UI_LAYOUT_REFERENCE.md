## Responsive Behavior

### Desktop (> 1024px)
- Full layout with sidebar
- Maximum search bar width: 720px
- Grid view: 4-5 columns

### Tablet (768px - 1024px)
- Sidebar collapses to icons only
- Search bar width adapts
- Grid view: 3 columns

### Mobile (< 768px)
- Hamburger menu for sidebar
- Full-width search bar
- List view forced
- Grid view: 2 columns

## Interactive States

### Hover States
- **Buttons**: Background changes to #f1f3f4
- **File rows**: Background changes to #f8f9fa
- **Nav items**: Background changes to #f1f3f4

### Active States
- **Nav item**: Blue background (#e8f0fe), blue text (#1a73e8)
- **View toggle**: Blue background (#e8f0fe), blue icon
- **Starred**: Yellow star (#fbbc04)

### Focus States
- **Search bar**: Subtle shadow
- **Buttons**: Outline for accessibility

## Spacing System

```
xs:   4px   - Between close icons
sm:   8px   - Button padding, item gaps
md:   12px  - Icon + text gap
lg:   16px  - Card padding
xl:   24px  - Section padding
xxl:  48px  - Large spacing
```

## Border Radius

```
Small buttons: 50% (circular)
Regular buttons: 4px
Cards: 8px
Search bar: 8px
New button: 24px (pill shape)
Nav items: 0 24px 24px 0 (right side only)
```
